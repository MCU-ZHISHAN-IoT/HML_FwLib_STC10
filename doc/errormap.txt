#Description : record error code which defined by HML

 - 0x01
    User need define input clock frequency via macro __CONF_FRE_CLKIN in conf.h.

 - 0x02
    HML gets a unknow or unsupported MCU model, please change value of macro __CONF_MCU_MODEL in conf.h.

 - 0x03
    Unspecify MCU model via macro __CONF_MCU_MODEL in conf.h.

 - 0x04
    Invalid operation in configuration of compilation macro. Macro __CONF_COMPILE_RCC depends on __CONF_COMPILE_UTIL.

 - 0x05
    Invalid operation in configuration of compilation macro. Macro ___COMPILE_UART___ depends on ___COMPILE_RCC___, ___COMPILE_TIM___, ___COMPILE_UTIL___.

 - 0x06
    User need update their own SDCC toolchain, because HML_FwLib_STC10 requires v3.6.0 or later version.
